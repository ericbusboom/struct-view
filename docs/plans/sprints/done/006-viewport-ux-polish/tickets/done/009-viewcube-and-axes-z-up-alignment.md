# ViewCube and axes Z-up alignment

## Description

Ensure the ViewCube face labels and axis arrows are consistent with the Z-up
coordinate system used throughout the app.

## Acceptance Criteria

- [ ] Axis arrows: Z=up, Y=front-to-back, X=left-to-right
- [ ] ViewCube face labels match the Z-up convention
- [ ] Clicking a ViewCube face animates the camera to the correct orientation with the correct up vector
- [ ] ViewCube orientation stays in sync with turntable orbit controls
