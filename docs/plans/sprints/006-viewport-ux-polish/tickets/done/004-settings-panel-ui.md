# Settings panel UI

## Description

Add a settings panel (gear icon in the sidebar or toolbar) that exposes the
settings store values to the user.

## Acceptance Criteria

- [ ] Gear icon or similar affordance opens a settings panel
- [ ] Imperial / metric toggle for unit system
- [ ] Snap grid size input (e.g., 1 ft, 0.5 m, 1/4 in)
- [ ] Grid line spacing input
- [ ] Work plane size input
- [ ] Changes take effect immediately (no save button needed)
- [ ] Panel can be dismissed easily (click outside, Escape, or close button)
